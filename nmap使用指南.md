# nmap 使用方法

nmap 是一个强大的网络端口扫描工具。

## 基本扫描

```bash
nmap 192.168.1.1          # 扫描单个IP
nmap example.com          # 扫描域名
nmap 192.168.1.1-20      # 扫描IP范围
nmap 192.168.1.0/24      # 扫描子网
```

## 常用选项

```bash
nmap -p 80,443 192.168.1.1       # 指定端口
nmap -p 1-1000 192.168.1.1      # 端口范围
nmap -sV 192.168.1.1            # 版本检测
nmap -O 192.168.1.1             # 操作系统检测
nmap -A 192.168.1.1             # 全面检测（OS+版本+脚本）
nmap -sC 192.168.1.1            # 默认脚本扫描
```

## 扫描类型

```bash
nmap -sS 192.168.1.1            # SYN扫描（需root）
nmap -sT 192.168.1.1            # TCP连接扫描
nmap -sU 192.168.1.1            # UDP扫描
```

## 输出

```bash
nmap -oN output.txt 192.168.1.1  # 保存正常格式
nmap -oX output.xml 192.168.1.1  # 保存XML格式
nmap -oA output 192.168.1.1     # 保存所有格式
```

## 常用示例

```bash
# 扫描常见端口+版本检测
nmap -sV 192.168.1.1

# 扫描子网指定端口
nmap -p 22,80,443 -sV -oN scan.txt 192.168.1.0/24

# 全面检测
nmap -A 192.168.1.1
```

> 注意：扫描非自有权益网络需授权。
